Take Notes

I built a simple notes app using React, Django, PostgreSQL, Docker, Docker Compose, AWS, and JWT. The app allows users to create, edit, and delete notes. It also supports searching for notes and sharing notes with others. The app is built on a React frontend and a Django backend. The database is PostgreSQL and it is hosted on AWS. The app is authenticated using JWT. I used Docker and Docker Compose to make it easy to deploy the app. I also used AWS to host the app so that it is available to users all over the world. I am proud of this project because it is a fully functional app that I built from scratch. I learned a lot about React, Django, PostgreSQL, Docker, Docker Compose, AWS, and JWT in the process.
